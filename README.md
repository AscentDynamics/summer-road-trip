# Summer Road Trip 2026

A mobile-friendly trip dashboard for 6 friends, 2 cars, 25 days, and 22 national parks.

**Live site:** https://ascentdynamics.github.io/summer-road-trip/

---

## Features

- **Today** — shows the day's stops with one-tap Google Maps navigation. GPS-powered buttons to find gas stations, grocery stores, Walmarts, and campgrounds nearby.
- **Budget** — log expenses by category (gas, food, camping, park fees). Auto-splits the total per person. Gas estimator lets you plug in each car's MPG and estimated miles to forecast fuel cost for the whole trip.
- **Itinerary** — all 39 stops across 25 days. Check off each stop as you visit it. Tracks parks visited, states hit, and total stops done.
- **Settings** — set the trip start date (drives the day counter and all dates), adjust the number of people splitting costs.

## Usage

No install. Open the link on any phone browser. Each person's data (expenses, visited stops) saves locally to their own device via `localStorage`.

To update the site after making changes:

```bash
git add index.html
git commit -m "describe your change"
git push
```

GitHub Pages redeploys automatically within ~60 seconds.
